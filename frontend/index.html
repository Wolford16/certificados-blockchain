Registro Manual en Blockchain

Para registrar su certificado de forma inmutable, siga estos pasos:

  1. Abra Remix IDE: remix.ethereum.org
  2. Cargue el Contrato: Pegue el código de CertificateRegistry.sol y compílelo.
  3. Despliegue: Vaya a la pestaña "Deploy & Run Transactions" y despliegue el contrato.
  4. Registre el Hash: Busque la función issueCertificate, pegue el hash copiado y ejecute la transacción.
  5. Verifique: Use la función verifyCertificate con el mismo hash para confirmar el registro.
